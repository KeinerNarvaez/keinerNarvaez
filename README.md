# Aprendizaje continuo y autodidacta

>Soy alguien que le gusta desarrollar programas pequeños con el objetivo de comprender el funcionamiento de los lenguajes de programación, las ventajas del lenguaje o simplemente su sintaxis.

>Me encanta ver videos tutoriales y aplicar lo aprendido de ellos, me fascina lo que hoy en día una persona puede lograr sentada frente a una computadora: herramientas, aplicaciones, formas de solucionar problemas y proyectar esas ideas en código puro y limpio.

Cada proyecto es un nuevo mundo por explorar, es una oportunidad para mejorar mis habilidades, aprender de mis errores y a menudo me pregunto hasta dónde podré llegar como desarrollador.
